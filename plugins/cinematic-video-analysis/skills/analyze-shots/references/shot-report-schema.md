# Shot analysis report

Use this order:

1. **Source** — filename, duration, dimensions, frame rate, codec.
2. **Segmentation** — detector threshold, shot count, cuts per minute, mean and median shot length, confidence caveats.
3. **Shot table** — `ID | In | Out | Duration | Transition | Verified visual summary | Keyframe`.
4. **Pacing** — longest/shortest sections, clusters of rapid cutting, sustained takes.
5. **Review notes** — suspected false cuts, missed transitions, unreadable frames.

For each visual summary, describe only visible subjects, action state, setting, framing, and dominant light/color. Keep interpretation separate from observation.
